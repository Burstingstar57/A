The purpose of Project Xena

Increase awareness

One of the main goals of Project Xena is to raise awareness about A.I. Increasing the amount of daily activities we can complete with the help of A.I. A.I is a very usefull resource everyone should be informed about.

Passive Income

Each holder of a Xena NFT will automatically be entered in the monthly giveaway with a prize of 1 ETH. The giveaway is only for memebers of the Xena community and you must own a Xena NFT to participate.

Free Courses

Project Xena will support all Xena NFT holders with free courses to learn more about any digtal art, A.I, and multimedia course. Xena NFT holders will be able to choose 1 course per NFT, even if buying second-hand.

Charity

Project Xena will host a variety of events all generating money for charity. Special games will be hosted that Xena NFT holders can play, all A.I based. These events will be free for all but NFT holders will have special bonuses.

Recent Projects

Phase 1

Release

Phase 1 of Project Xena is the release of the Project Xena NFT and the opening of the mint process. 1,000 NFTs will be able to be minted for 0.06 Ethereum each. 2 Xena NFTs will be allowed per wallet. The other 111 remaining NFTs will be saved for collaborations, giveaways, and future special drops for the Project Xena community.

Phase 2

Loyalty Reward

Phase 2 will occur when 50% of the NFTs have been minted. The owners of the first 500 minted Xena NFTs will be entered in a giveaway to earn one special animated NFT for free.

Phase 3

First Giveaway

Phase 3 will occur when all 1,000 NFTs have been sold. The first monthly giveaway will begin. All holders will be put into the giveaway automatically by NFT number. One random NFT number will be selected and that will be the winner of the giveaway. The winner reveal will be done live on discord and twitter. Many announcments will be made prior so that everyone will know and be able to attent the event.

Phase 4

Xena v2

When phase 4 occurs Project Xena v2 will release. There will be 2,222 mew NFTs in this drop. The v2 series will contain different themed and styled NFTs. The v2 series will also have seperate exclusive content. There will be a completely new whitelist and whitelist and whitelist requirements for this drop. People that missed the original mint will have a chance to mint in v2.

Phase 5

Increase/Event

A special event will be hosted for Xena NFT holders. The event will be selected based on which has the highest community vote. After the event takes place monthly giveaway prizes will be increased by 2%. This is not the last step in the roadmap, this will be updated as time goes on.
